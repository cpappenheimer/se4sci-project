# se4sci-project

# Installing the package in a virtual environment
```
python3 -m venv venv
. venv/bin/activate
pip install -e .
```

# Running the example
```
python src/examples/example.py
```

# Running tests
```
nox -s tests
```

# Running tests with log output
```
nox -s tests -- -rP
```
