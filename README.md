# se4sci-project

# Running the example
```
pixi run python src/examples/example.py --file <path to input data file> --tree-name <name of tree>
```

# Running pre-commit
```
pre-commit run -a
```

# Running tests
```
pixi run -e test pytest
```

# Running tests with log output
```
pixi run -e test pytest -rP
```
